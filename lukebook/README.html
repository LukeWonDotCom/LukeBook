--Requirements--
  • Mysql Server
  • Phpmyadmin (Must be this)
  • Php server
--Installation--
  1. Unzip/move the files into a folder in your webserver
  2. Go to phpmyadmin and import lukebook.sql to your desired database
  3. Open the folder and edit db.php
  4. Replace 'Your-hostname', 'Your-Username', 'Your-Password' and 'Your-DB' with your real Database info (Only Mysql database will work)
  5. Go to 'your-webserver/where-ever-you-put-the-files/index.php' (Defult Username: root) (Defult Password: root)
